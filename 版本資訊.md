## 📞 版本升級指南

### 從 v3.0 升級到 v3.1

**步驟：**
1. 匯出 JSON 備份（舊版）
2. 下載 v3.1 版本檔案
3. 重新啟動後端服務（如果有更新）
4. 開啟 v3.1 HTML 檔案
5. 匯入 JSON（系統會自動補全預期結果欄位）
6. 手動填寫每個測試案例的預期結果

**相容性：**
- ✅ 舊資料自動相容
- ✅ 預期結果自動補空白
- ✅ 其他欄位完整保留

### 從 v2.0 升級到 v3.1

**步驟：**
1. 匯出 JSON 備份（v2.0）
2. 下載 v3.1 版本所有檔案
3. 重新啟動後端服務（**必須**）
4. 開啟 v3.1 HTML 檔案
5. 匯入 JSON
6. 重新上傳截圖和測試資料檔案
7. 填寫預期結果欄位

**注意：**
- ⚠️ 需要重新上傳檔案（v2.0 檔案只在瀏覽器）
- ⚠️ 必須更新 backend_api.py
- ✅ 文字資料完整保留
